# Lumina — App Store Assets

This directory contains all assets needed for App Store and Google Play submissions.

## Required Screenshots

### iPhone (6.9" display — iPhone 16 Pro Max)
- `screenshot-01-home.png` — Home screen with AI greeting + module grid
- `screenshot-02-journal.png` — Journal write panel with AI prompt
- `screenshot-03-wellness.png` — Wellness charts (mood trend + sleep)
- `screenshot-04-mind.png` — Memory cards with search
- `screenshot-05-life.png` — Task list with AI focus tip

### iPad Pro (13" — optional but recommended)
- Same 5 screens in landscape format

### Android (phones)
- Same 5 screens at 1080×2400px (or device frame overlay)

## Required Graphics

| Asset              | Size            | File                    |
|--------------------|-----------------|-------------------------|
| App Icon           | 1024×1024px     | `icon-1024.png`         |
| App Icon (Android) | 512×512px       | `icon-512.png`          |
| Feature Graphic    | 1024×500px      | `feature-graphic.png`   |
| Splash Screen      | 2048×2048px     | `splash.png`            |

## Listing Content

- **description.txt** — Full app description (App Store & Play Store)
- **keywords.txt** — ASO keywords + subtitle + short description
