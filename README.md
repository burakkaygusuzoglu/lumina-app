# Lumina — Life OS

> Your mind, amplified.

AI-powered personal life operating system. Notes, wellness, vault, tasks, and journal — all in one place.

## Tech Stack
- Backend: FastAPI + Python
- Database: Supabase (PostgreSQL)
- AI: Claude API (Anthropic)
- Frontend: React + TypeScript
- Mobile: React Native (Expo)
